# Biblia online

## **Instalacja pakietów**

* Wpisz w terminalu po kolei następujące komendy:

``` javascript
npm init
```

``` javascript
npm install express
```

## **Uruchomienie serwera** 

* Wpisz w terminalu tę komendę:
   
``` javascript
node server.js
```
